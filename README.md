# :woman: - mullen
Renames files for Ms. Mullen's English Class Standard
## Configuration
Specify document directory and last name in ~/.mullenrc
````
[Mullen]
Document_Dir = $DOCUMENT_DIR
Last_Name = $LASTNAME
````
For example:
````
[Mullen]
Document_Dir = /home/john/documents/school/English
Last_Name = smith
````
